La prima webcam pubblica del paese in provincia di Torino

Webcam #Cumiana nasce come progetto ad inizio maggio 2015, un progetto realizzato in economia che utilizza hardware "di recupero" e software Open Source.

Tecnologia

Il progetto è stato per ora realizzato in completa economia.

Hardware

  • Samsung Galaxy S GT-i9000
  • Debian VM

Software

  • Tasker app per Android
  • Script eseguito da tasker per scatto foto ed invio tramite protocollo HTTP
  • Script PHP per il salvataggio delle immagini su server Debian

Vuoi installare anche tu una webcam?

Non è difficile, come hai visto è sufficiente un vecchio telefono smartphone, un'applicazione che puoi trovare su Google Play, ed uno spazio web.
A breve rilascerò il codice utilizzato per far funzionare il tutto (licenza MIT), aspetto solamente di portare il progetto in configurazione stabile.

Comunque indicativamente funziona così:

l'applicazione Tasker permette di programmare il proprio telefono ed eseguire operazioni pianificate nel tempo (e molto altro)

Ogni 10 minuti Tasker lancia una sequenza di azioni:

  • scatto foto
  • salvataggio foto su file
  • lettura file binario e codifica in base64
  • url_encode del precedente risultato
  • invio richiesta HTTP, metodo POST al server inviando come parametro testuale il contenuto del file codificato

A questo punto il server riceve una richiesta HTTP, legge la variabile POST inviata, la decodifica, controlla che effettivamente si tratti di un'immagine, ed in caso affermativo salva una nuova immagine su filesystem, dopo aver aggiunto una scritta con data ed ora.
